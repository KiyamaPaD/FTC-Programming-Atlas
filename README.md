# FTC Programming Atlas

Interactive documentation map for FTC programming concepts.

## What it is
This project is a visual atlas for FTC programming knowledge:
- TeleOp
- Autonomous
- PID / Control
- Localization
- Pathing
- Troubleshooting

The goal is to help future team generations learn faster through a node-based documentation system.

## Current features
- Interactive node map
- Full-screen documentation panels
- Editable relations between topics
- Undo / redo
- Local save system

## Future plans
- Supabase sync
- Authentication for collaborators
- Public online deployment
- Better team documentation workflow
